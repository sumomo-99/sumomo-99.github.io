[bash] bashで2048ゲーム¶

bash2048 はbashで動く 2048ゲーム です。

  • インストール＆実行
  • オプション
  • 操作方法

インストール＆実行¶

$ cd /usr/local/games/
$ git clone https://github.com/mydzor/bash2048.git
$ cd bash2048
$ ./bash2048.sh

オプション¶

-b <数値>¶

盤面のサイズを3～9の範囲で変更します。 デフォルトは4です。

-t <数値>¶

勝利までの点数を指定します。 デフォルトは2048です。

-l <ファイル名>¶

デバッグログを出力します。

-r¶

セーブしたゲームを読み込みします。 ゲーム中に Control-c で終了するときにセーブできます。

-h¶

ヘルプを表示します。

オプションをつけた実行例

盤面のサイズを3×3に変更したい場合:

$ ./bash2048.sh -b 3

セーブデータを読み込む場合:

$ ./bash2048.sh -r

操作方法¶

カーソルキーで操作します。 Cotorl-c で終了します。
